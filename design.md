# Lume.ai Design System

## 1. Vision
High-fidelity autonomous research. The design should feel like a premium, distraction-free scholarly workbench. It combines the aggressive technical edge of Mistral (colors, type) with the clarity and accessibility of Elicit.

## 2. Color Palette
- **Primary Background:** `#F5F2ED` (Mistral Beige) - Used for the "paper" feel.
- **Secondary Background:** `#FFFFFF` (White) - Used for cards and active panels.
- **Accents:** `#FF511C` (Mistral Orange) - Used for call-to-actions, progress indicators, and highlights.
- **Text:** `#000000` (Midnight Black) - Primary text.
- **Muted Text:** `rgba(0, 0, 0, 0.4)` - Secondary metadata and descriptions.
- **Borders:** `rgba(0, 0, 0, 0.05)` - Subtle hairline separators.

## 3. Typography
- **Headlines:** `font-serif` (EB Garamond / Space Grotesk Semibold) - Communicates authority and precision.
- **Body:** `font-sans` (Inter / Geist) - For general UI and readable content.
- **Technical/Logs:** `font-mono` (IBM Plex Mono) - For agent status, timestamps, and data extraction tags.

## 4. UI Components (Sharp & Professional)
- **Cards:** White background, `rounded-none`, solid `border-[0.5px]` in `mistral-black/10`, flat or very subtle shadow.
- **Radius:** Zero (sharp corners) for all main containers and inputs.
- **Buttons:**
  - *Primary:* Solid Black, White text, `rounded-none`, smaller padding (px-4, py-2).
- **Inputs:** `rounded-none`, hairline border.

## 5. Dashboard Principles
1. **Compact Density:** Smaller text and less padding to show more information.
2. **Sharp Boundaries:** No rounded corners; everything is rectangular and precise.
3. **Dynamic Feedback:** Use typing animations for empty states to guide the user.
